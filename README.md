# Project Title
MotoGauge

## Description
MotoGauge is a tool built to provide accurate valuations for motorbikes across the NSW market. The tool is based on real-world data and insights rather than estimations or old datasets used by other tools currently available. It takes into account factors that newer generations of riders consider, which older websites often overlook.


## Features
MotoGauge uses a range of information that can affect the base value of a motorbike, such as modifications, aftermarket features, restriction status, registration status, and more to provide an accurate estimate.


## Technologies Used
The website uses CSS, HTML, and JavaScript.


## Project Structure
The first page is a starter page that gives users a brief introduction when opening the tool. This is designed to welcome the user and ensure that the tool is suitable for their intended use. Once they click "Start Valuation", they are then taken to the valuation tool where they can fill out the required information.


## Usage
This tool can be used for:
- Purchase of a motor vehicle
- Sale of a motor vehicle at market value
- Planning a sale with the goal of making a profit
- Depreciation calculation

## Future Improvements
- A possible verification system where verified users can upload their own collected data
- Increased database of manufacturers and models
- Calculations for larger market regions
